[js-rational](http://aureooms.github.io/js-rational)
==

<img src="https://upload.wikimedia.org/wikipedia/commons/1/17/Number-systems.svg" width="864">

Rational numbers code bricks for JavaScript.

```js
let [ a , b ] = add( 1 , 10 , 1 , 5 ) ;
a / b === 3 / 10 ; // true
```
